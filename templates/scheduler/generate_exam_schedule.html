Setup & Requirements
What the algorithm does:
  • Schedules exams for all courses in the selected college
  • Assigns appropriate rooms based on class size
  • Ensures no conflicts between classes and rooms
  • Assigns multiple invigilators per exam session
  • Balances invigilator workload across all exams
Requirements (Setup in Admin):
  • Exam Dates: Create exam dates in Timetable → Exam dates
  • Time Slots: Create time slots with "Is exam slot" checked
  • Proctors: Mark lecturers as proctors (Is proctor = True)
  • Rooms: Ensure rooms have sufficient capacity
